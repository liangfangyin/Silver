**1、生产者**

```
var publisher = new KafkaMQPublisher<TValue>();
while (true)
{
	//生产信息
    bool ispublish = publisher.Publish("通道名称",TValue,等待时间毫秒);
    bool ispublish =await publisher.PublishAsync("通道名称",TValue); 
}
//关闭生产者
publisher.Close();
```
